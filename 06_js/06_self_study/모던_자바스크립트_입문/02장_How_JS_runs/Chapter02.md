# :keyboard: 02장. 프로그램의 작성법과 실행법

<br>

## 1. 자바스크립트 작성법

- 코드 입력할 때 주의 사항
  - 공백 문자를 입력할 때는 `Space`를 사용한다.
  - 들여쓰기에는 `Tab`을 사용한다.
  - <b><u>세미콜론 입력을 잊지 않는다.</u></b>
  - 대문자와 소문자를 확실하게 구분하자.
- 저장할 때 주의 사항
  - 파일 이름 끝에 확장자를 붙인다.
  - 파일의 문자 인코딩은 UTF-8로 설정한다.

<br>

## 2. 자바스크립트 프로그램 실행법

- 구글 Chrome hrome 개발자 도구 이용하기
- 자바스크립트 코드를 HTML 문서에 삽입하여 웹 브라우저로 실행하기
- Node.js의 대화형 모드로 실행하기
- Node.js의 파일을 읽어 들여 실행하기
